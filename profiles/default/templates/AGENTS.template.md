## AMP-OS Skill Triggers

Use these phrases to activate AMP-OS workflows:

- **"plan product"** - Start product roadmap planning
- **"shape spec"** - Research and gather requirements
- **"write spec"** - Create feature specification
- **"create tasks"** - Break spec into implementation tasks
- **"implement tasks"** - Execute task implementation workflow
- **"verify implementation"** - Run final verification checks
